# Tea Leaf Disease Spot Dataset

This repository contains a tea leaf image dataset for tea disease spot analysis.

## Dataset Structure

```text
Tea-leaf-disease-spot-dataset-
├── bad/      # diseased tea leaf images
├── good/     # healthy or visually normal tea leaf images
├── manifest.csv
└── README.md
```

## Classes

| Folder | Description |
|---|---|
| `bad` | Tea leaf images with visible disease spots or symptoms |
| `good` | Healthy or visually normal tea leaf images |

## File Format

All images are stored in JPG format.

## Notes

This dataset is released for research use in tea leaf disease image analysis. Please cite or acknowledge this dataset if it is used in academic work.
